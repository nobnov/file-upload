# File Upload
## 1.モジュールをインストール
    $ pip install requests

## 2.URLを指定
    url = 'YOUR DOMAIN'
    (XXXXX.com, XXXXX.co.jp)

## 3.ファイルを指定 (post.pyと同じディレクトリ内)
    image = '001.jpg'
    (002.png, 003.txt)

## 4.実行コマンド
    $ python post.py
